# subemeapp

_Subeme App is a web app to upload files._

## Funcionalidades:

* CRUD File
* CRUD File Group
* S3 Upload File
* S3 List
* S3 Delete file in replace
* ZIP documents in one file (LocalHost and S3 works)
* API CRUD request 'api/'
